# ☕ My Java Journey

A structured repository documenting my progression from Java beginner to building larger, real-world style applications.

The goal of this repository is not simply to collect finished projects, but to demonstrate continuous learning, refactoring, problem-solving, and the gradual development of professional software engineering practices.

---

## 🎯 Repository Goals

* Build a strong foundation in Java
* Learn object-oriented programming through practical projects
* Develop clean architecture and separation of concerns
* Practice defensive programming and validation
* Improve code quality through refactoring
* Learn industry-standard tools and workflows
* Create a portfolio of progressively more advanced projects

## 📚 Learning Path

### 01 - Basics

Core Java fundamentals:

* program structure
* variables and data types
* arithmetic operations
* user input with Scanner
* conditional logic
* loops

---

### 02 - Interactive Logic

Building interactive console applications:

* validation loops
* defensive programming
* menu-driven workflows
* structured control flow

---

### 03 - OOP Foundations

Introduction to object-oriented programming:

* classes and objects
* constructors
* encapsulation
* object state management
* basic architecture concepts

---

### 04 - Collections

Working with groups of objects:

* ArrayList
* Comparable
* Comparator
* sorting strategies
* collection management

---

### 05 - Projects

Larger applications that combine multiple concepts into complete systems.

#### [Student Manager System](./05-Projects/01-StudentManagerSystem) //FIIIIIIIXX

A console-based student management application featuring:

* layered architecture
* validation
* file persistence
* CRUD operations
* search and sorting functionality

#### [JSON to HTML Parser](./05-Projects/02-JsonToHtmlParser)

A Maven-based application that transforms structured JSON into HTML through a recursive rendering pipeline.

Key topics:

* recursion
* tree structures
* JSON deserialization
* rendering systems
* JUnit testing
* command-line applications
